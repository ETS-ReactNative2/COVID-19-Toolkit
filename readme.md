# COVID -19

An essential toolkit to fight COVID-19

## Getting Started

The app is based on React Native (Expo)

### Dependencies

Expo CLI (optional)

```
$ npm install -g expo-cli
```

### Installing

Clone the repo then use

```
$ npm install
```

To start the app use

```
$ npm start
```
